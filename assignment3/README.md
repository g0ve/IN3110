# How to run scripts

## 3.1 wc (3 points)
How to run wc.py - Wordcounter
- File paramters:
  - filename.txt
  - '*' - all files in directory
  - *.py - all python files in directory
```
$ python wc.py [file]

```
The character counter counts '\n'. I decided to let i be like that, but if its not correct, its a easy fix with '-1'
